# Demo Web Crypto API

## Build and start

```sh
yarn install
yarn build-client
yarn startt
```

## Dev mode

```sh
yarn install
yarn start-server &
yarn start-client &
```

## Usage

go to http://localhost:8080 and log in with bob@foo.bar/password or alice@foo.bar/password. Then you can write messages to other users (if they logged in at least once).
